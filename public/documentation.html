theMovieBook

We will not drop the "the"

At some point, this will be an API that interacts with a database of movies (including actors, directors, genres, etc.) allowing users to create an account, create a list of favorites, and search/sort/filter by whatever they want.

FAVORITES
Retrieve list of user favorites /users/[username]/favs GET None A JSON object containing data for all user favorites.
Add movie to user favorites list. /users/[username]/favs POST None A plaintext message indicating movie was successfully added to favorites.
Delete movie from user favorites list. /users/[username]/favs/[movieId] DELETE None A plaintext message indicating movie was successfully removed from favorites.
